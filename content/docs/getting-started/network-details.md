# Network Details

Use the following details to add the MuriData Testnet to your mobile or browser wallet (MetaMask, Core, Rainbow, Trust Wallet, etc.).

## Wallet Configuration

| Field | Value |
|---|---|
| **Network Name** | MuriData Testnet Alpha |
| **Chain ID** | `97981` |
| **Currency Name** | MuriCoin |
| **Currency Symbol** | MURI |
| **Decimals** | 18 |
| **RPC URL** | `https://testnet-rpc.muri.moe/ext/bc/inP2vNhcVSABGmq39UHwuB9tDxUUWp3g6gpRwdE6TqtAtAWmu/rpc` |
| **Block Explorer** | `https://testnet-explorer.muri.moe` |

## Adding the Network

### MetaMask (Mobile & Browser Extension)

1. Open MetaMask and tap **Settings** > **Networks** > **Add Network**.
2. Select **Add a network manually**.
3. Enter the values from the table above.
4. Tap **Save**.

### Core Wallet

1. Open Core and go to **Settings** > **Networks**.
2. Tap **Add Custom Network**.
3. Fill in the fields using the values above.
4. Tap **Save**.

### Other Wallets

Most EVM-compatible wallets (Trust Wallet, Rainbow, Rabby, etc.) support adding custom networks. Look for an **Add Custom Network** or **Add RPC** option in the wallet settings and enter the values from the table above.
